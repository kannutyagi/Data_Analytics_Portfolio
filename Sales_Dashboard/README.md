# Superstore Sales Dashboard

## Overview
This project is an interactive Superstore Sales Dashboard created using Power BI.

## Features
- Sales performance analysis
- Category-wise sales insights
- Monthly sales trend analysis
- Product-wise sales analysis
- Interactive filters and KPIs

## Tools Used
- Power BI
- Excel / CSV
- Data Cleaning
- Data Visualization

## Key Insights
- Technology category generated the highest sales.
- Monthly sales trends show fluctuations across different order periods.
- Top-selling products contributed significantly to overall revenue.
