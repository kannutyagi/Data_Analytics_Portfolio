# HR Analytics Dashboard

## Overview

This project is an interactive HR Analytics Dashboard created using Power BI.

## Features

* Employee attrition analysis
* Salary analysis
* Age group analysis
* Job role insights
* Workforce trends

## Tools Used

* Power BI
* Excel / CSV
* Data Cleaning
* Data Visualization

## Key Insights

* Highest attrition occurs among employees aged 26–35.
* Laboratory Technicians and Research Scientists show higher attrition rates.

## Files Included

* Power BI dashboard (.pbix)
* Dataset (.csv)
* Dashboard screenshot
